# Featurama .NET MAUI SDK

Official .NET MAUI SDK for [Featurama](https://featurama.io) - Feature Request Management for Mobile Apps.

> Status: Coming soon for MVP. React Native / Expo is currently the production-ready SDK.

- Flutter: Coming soon
- Swift: Coming soon
- Android: Coming soon
- .NET MAUI: Coming soon

API and packaging details may change before general availability.

## MVP Recommendation

For production integrations today, use `@featurama/react-native` in React Native / Expo apps.
